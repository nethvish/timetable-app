About app


The Timetable App project was created to give our colleagues a smooth experience with their daily lecture timetables. Apart from the typical chart-type tables those were published, this web app shows the user what lectures are available next, what lectures are ongoing now, and more features.

Features ✨

  • 👉 View all lectures as cards in order
  • 👉 Highlights the ongoing lecture
  • 👉 View timetables of other days
  • 👉 Add to homescreen to use like a mobile app

Tech stack 📦

The app is made of complete vanilla Javascript. The Data is stored in a JSON file which has obvious limitations, but an improvement will be introduced soon.
